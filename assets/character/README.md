# Character Parts

The outfit preview uses these 1024x1024 transparent PNG parts.

- `body/head.png`: face, ears, hair, neck
- `body/arms.png`: arms behind clothes
- `body/hands.png`: hands in front of sleeves/clothes
- `body/shorts-legs.png`: bare legs shown when shorts are selected

Keep the 1024x1024 transparent canvas and the same drawing position when editing so the parts stay aligned.

Clothing parts also use 1024x1024 transparent PNG files.

- `clothes/tops/`: tops with visible sleeves
- `clothes/tops-under-jumper/`: tops hidden under a jumper
- `clothes/hoods/`: hoodie back and front hood layers
- `clothes/bottoms/`: pants and shorts
- `clothes/shoes/`: shoes
- `clothes/outers/`: jumpers and vests

When editing clothes, keep the canvas size and the character alignment unchanged. The transparent empty space is part of the positioning system.

Season backgrounds are 1024x1024 PNG files.

- `backgrounds/summer.png`: short-sleeve outfits
- `backgrounds/spring.png`: light layered outfits
- `backgrounds/autumn.png`: light layered outfits
- `backgrounds/winter.png`: jumper or hoodie with warm vest outfits
